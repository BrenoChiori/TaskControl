# 📝 Task Control

Aplicação simples de gerenciamento de tarefas desenvolvida com **React**, com o objetivo de praticar conceitos fundamentais da biblioteca.

🔗 **Acesse o projeto online:**
https://task-control-tau.vercel.app/

---

## 🚀 Sobre o projeto

O **Task Control** é um pequeno aplicativo que permite gerenciar tarefas do dia a dia de forma simples e eficiente.

Este projeto foi criado com foco em **aprendizado prático de React**, explorando conceitos essenciais como:

* Componentização
* Gerenciamento de estado com `useState`
* Manipulação de listas
* Persistência de dados com `localStorage`

---

## ⚙️ Funcionalidades

* ✅ Criar novas tarefas
* 📋 Listar tarefas
* ❌ Remover tarefas

---

## 🛠️ Tecnologias utilizadas

* React
* JavaScript (ES6+)
* HTML5
* CSS3
* UUID (para geração de IDs únicos)
* Vercel (deploy)

---

## 💾 Persistência de dados

As tarefas são armazenadas localmente no navegador utilizando o **localStorage**, garantindo que os dados não sejam perdidos ao recarregar a página.

---

## 🎯 Objetivo

Este projeto foi desenvolvido com o intuito de:

* Praticar os fundamentos do React
* Entender melhor o fluxo de dados em aplicações frontend
* Construir uma base sólida para projetos mais complexos
